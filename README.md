# Lelylan Site

Source code for [lelylan.com](http://lelylan.com).


## Requirements

Devices API is tested against MRI 1.9.3.


## Installation 

    $ git clone git@github.com:lelylan/site.git && cd site
    $ gem install bundler
    $ bundle install 
    $ foreman start
