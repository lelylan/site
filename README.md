# betterspecs.org

Source code for [betterspecs.org](http://betterspecs.org)

## Requirements

RSpec Best Practices website is tested against MRI 1.9.3.


## Installation

    git clone git@github.com:lelylan/betterspecs.git
    cd betterspecs
    bundle install
    nanoc3 aco
